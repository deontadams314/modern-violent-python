## ğŸš€ Usage

```bash

```

### Arguments:
| Flag | Description |
|------|-------------|
| `--t` | Target IP or hostname |
| `--p` | Comma-separated ports (e.g. `--p 22,80,443`) |
| `--r` | Port range (e.g. `--r 1-1024`) |
| `--threads` | Number of threads to scan in parallel (default: 100) |
| `--output` | Save results to a text file |
